# This is instructions for automation installation 

- Set Your moniker 
```bash
MONIKER="your moniker name"
```

- Clone repositorie
``` 
git clone https://github.com/Zlkcyber/selfchaind.git
```
- Make it executable
```
cd selfchaind
```
```
chmod +x auto_install.sh
```

- AND RUN THE SCRIPT
```
./auto_install.sh
```
